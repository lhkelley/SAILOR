Once there, run the following commands to load preset modules (you have to do this everytime you start a new session):

```console
module load conda
module load apptainer
```
To check that miniconda is loaded, run:

```console
module list
```

## Installation and getting set up 
These are common packages for RNA-seq analysis (might not be relevant for what you're doing, this is just for general notes on creating conda environments).

You only need to run these commands once:

```console
conda install -c bioconda sra-tools
```
Now we'll create a conda environment and load it with the programs/packages we'll need:

```console
conda create -n rnaseq -c conda-forge -c bioconda entrez-direct sra-tools==3.1.1 fastqc seqtk star samtools subread
conda activate rnaseq
```
Note: Use ```conda list``` to check your version of sra-tools and change if needed in above line.

## Transferring files to your working space

We'll use the ```scp``` command to move the FASTQ files from my workspace to your workspace.
In my terminal, I'll run the command:

```
scp FILES.tar.gz [username]@quartz.uits.iu.edu:/N/slate/[username]/
```
And you'll have to type in your password on my laptop for the transfer to go through. Then you'll be able to find the directory as ```/N/slate/[username]```.
